This folder holds org-level context files. These are foundational documents that inform all vault work.

## Files

- `organization.md` -- Company structure, mission, products, audience, org chart
- `strategy.md` -- Company OKRs, department goals, quarterly priorities
- `brand.md` -- Tone, style guidelines, messaging, visual identity
- `services.md` -- Active services / products, revenue baseline, tech stack
- `icp.md` -- Ideal customer profile
- `pain-points.md` -- Customer pain points and objections
- `stakeholders.md` -- Vendors, partners, investors, advisors, key external contacts
- `infrastructure.md` -- Tool stack and integrations

## Rules

- Route stakeholder info (vendors, partners, investors, advisors) to `stakeholders.md`
- Route brand/voice/tone/messaging to `brand.md`
- Route strategy, OKRs, quarterly goals to `strategy.md`
- Route org structure, company info, products to `organization.md`
- Route ICP, customer profile to `icp.md`; pains to `pain-points.md`
- Route revenue lines, services, products to `services.md`
- Route tool stack changes to `infrastructure.md`
- These files are authoritative sources. Do not duplicate their content elsewhere.
- During weekly reviews, flag OKRs in `strategy.md` that have no active project.
